# Educational Vibration Analysis Website

## About This Project
This project is a simple website created to help students understand the basics of vibration analysis.  
The aim is to present the concepts in a clear and easy-to-learn format so that beginners can follow along without any difficulty.

## What This Project Contains
This repository includes:
• A main HTML file named `index.html`
• A `package.json` file
• Space for adding your own CSS, JavaScript, or images if you plan to improve the site in the future
